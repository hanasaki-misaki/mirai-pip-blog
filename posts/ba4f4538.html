2021年图床推荐

发表于2021-11-29|更新于2022-08-19
|阅读量:

前言

    本篇文章内容都可能随着时间推移而过时，不能保障100%的可用性。
    本篇内容纯个人意见并非集体，若有错误请指出，我会及时更改。
    制止目前为止（本文章发出时间）该项目、软件、程序、内容均为免费使用且可实施。

超星云盘

超星云盘是学习通的附属品，相信各位经历了2020的疫情后都不陌生。

web超星云盘登录入口

上传图片后点击图片会有一个预览窗口，这时图片是非常模糊的，那么如何改成原图并且获取链接当图床呢？

右键 → 在新标签页中打开图片 → 得到链接

举例：
原链接

http://imageproxy.chaoxing.com/0x0,q15,jpeg,s1CesE0NmKBdrVyWZP283veVdp0tib_5Q9VjL5dO7LjI/https://p.ananas.chaoxing.com/star3/origin/2918986589da31e4f127ef877a043887.png

将0x0,q15,jpeg,删除则获得原画质的图片，链接太长了？
看到那https协议的网址了吗？
正是这个
https://p.ananas.chaoxing.com/star3/origin/2918986589da31e4f127ef877a043887.png
但我个人不推荐这个搞，这个https有时候会掉，而且维护时间比较长，推荐用
https://imageproxy.chaoxing.com 开头比较好点因为云盘软件用的也是这个链接，相对维护https协议比较快，基本不会超过1天。
这个存储是国内的，速度没的说。

Github

Github，能来找到我这篇文章的各位应该都不陌生。
GitHub + PicGo + jsDelivr 也是个不错的选择虽然对部分地区的网络来说有点拉跨。
网上教程也多，这个我就不再赘述了。

幻天图床

幻天图床 - 萌图床 | 一个更专业的二次元图床中心
我能找到最老的记录是2015/07/16，个人感觉还是很不错的目前使用的是申请制度，能否成功纯看运气，审核时间也是看运气的，看看管理员什么时候闲着没事干了才会看一眼。

总结

超星云盘、Github、幻天图床都是不错的选择，但如果你要长期做下去，图床的不稳定因素绝对会有，而且不低，要稳定最好还是买个对象存储。
