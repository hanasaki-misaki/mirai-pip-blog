前言

使用我们提供的服务即视为同意本条款；如果您不同意本条款，请立即终止使用我们提供的任何服务。

API

Docs说明文档：API服务

     1.1 本项服务免费对外开放，使用时必须遵守所在地区及香港特别行政区的法律法规。

     1.2 不欢迎 版权站 / 广告站 / 采集站 等类型站点调用此服务。

     1.3 若您的站点对 API 调用量很大，建议您自建服务。

     1.5 若贵站引用 API 时返回错误，则说明已被加入黑名单，如需说明情况，请通过邮件 cloud2077@iysheng.com 联系。

     1.7 本 API 含有统计内容，如果拒绝统计可自行删除，如觉得不适请停止使用。
